== Description ==
Elum Send Activation Email plugin shows the list of inactive users who have used promocode and sends activation email(s) to them.
It comes with minimal configuration and works with latest version of Wordpress.

== Installation ==
1. Upload "elum-send-activation-emails" folder to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Thats it!
1. Done.
